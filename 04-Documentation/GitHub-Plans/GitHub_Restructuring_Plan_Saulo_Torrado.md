## 🎯 What is ProphetLabs?

ProphetLabs is an arbitrage engine that detects pricing discrepancies between prediction markets
in real time. It monitors Polymarket and Opinion Labs simultaneously via WebSocket streams,
identifies mathematical arbitrage opportunities (YES + NO spread capture), and provides
actionable alerts through a React dashboard and Telegram bot interface.

**Why it matters:** Prediction markets are fragmented. The same event can be priced differently
across platforms. ProphetLabs captures that alpha automatically.
