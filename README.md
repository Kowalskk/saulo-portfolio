## 🖼️ Preview

![Portfolio Preview](assets/gud.jpg)

## ⚡ Features

- **Glassmorphism UI** — frosted glass cards with backdrop-filter effects
- **Matrix background** — animated character rain (subtle, 3.5% opacity)
- **Responsive** — works on desktop and mobile
- **Single-page** — vanilla HTML/CSS/JS, no frameworks, no dependencies
- **Video showcases** — embedded project walkthroughs (ProphetLabs, Meteora, PRJX)
- **Terminal aesthetic** — monospace code blocks for the tools section

## 🏗️ Stack

| Layer | Tech |
|-------|------|
| Markup | HTML5 |
| Styling | CSS3 (custom properties, grid, flexbox, backdrop-filter) |
| Fonts | Syne, Space Mono, DM Sans (Google Fonts) |
| Hosting | Vercel |
| Domain | Custom (via Vercel) |
